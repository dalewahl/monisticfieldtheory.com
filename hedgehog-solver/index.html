About this calculation

The radial BVP

In the dense, screened regime of MFT, where the contraction field is approximately constant, the action reduces to a Skyrme-type chiral field theory for an SU(2) field U(r). The B = 1 hedgehog ansatz is:

U(r) = exp(i τ·r̂ f(r))

with f(0) = π (skyrmion at origin) and f(∞) = 0 (vacuum at infinity). The Euler-Lagrange equation for the radial profile is:

f''(r) = [sin(2f)(1 + sin²f/r²) − 2r f'(r) − sin(2f) f'(r)²] / (r² + 2 sin²f)

This is a nonlinear two-point BVP solved here by SciPy's solve_bvp (a 4th-order collocation method). The initial guess uses the analytic pure-Skyrme profile f₀(r) = 2 arctan(R_h / r).

What converges, and to what

A converged B = 1 hedgehog has four signatures:

  1. f(0) = π and f(rmax) ≈ 0 — boundary conditions met to machine precision.
  2. Virial balance E2/E4 = 1 — Derrick stability requires quadratic and quartic energy contributions to be equal at the soliton minimum. Standard preset achieves ~0.03%.
  3. Integer baryon number B = 1 — both the topological formula B = (f(0) − f(∞))/π and the integral B = -(2/π) ∫ sin²f f' dr give the same integer answer to 6+ decimal places.
  4. ε₀ = E2 + E4 ≈ 145.85 — the Adkins-Nappi-Witten reference value for the Skyrme energy in standard units.

Extracting (e, f_π) — and why it doesn't match the MFT prediction

With ε₀ and λ₀ (moment-of-inertia integral) from the BVP, two parameters of the Skyrme Lagrangian are determined by matching:

  • M_N = 938.3 MeV (nucleon mass)
  • M_Δ − M_N = 293.7 MeV (rotational excitation energy)

The result of this extraction is e ≈ 4.5 and f_π ≈ 107 MeV. The observed value is f_π = 186 MeV; the BVP-extracted value is therefore off by ~40%.

This is not a problem for MFT. The MFT-specific prediction of f_π comes from a different mechanism: the chiral-stiffness closure $f_\pi^2 = V''(\varphi_v)/4 = \delta = 1+\sqrt{2}$ in MFT units, which gives f_π = √δ × MFT_TO_MEV = 185.94 MeV (matching observed 186 to 0.03%).

The BVP shown here verifies the soliton structure of the MFT-Skyrme reduction (ε₀, B, virial balance), independent of the chiral normalisation. The phenomenological extraction of f_π by fitting M_N and M_Δ uses Skyrme model assumptions that differ from the MFT chiral-stiffness closure. Both calculations are consistent with each other; they're just computing different things. For the matching MFT prediction f_π² = δ, see Particles §6a or P8 of the corpus.
