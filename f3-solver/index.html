The Family-of-Three Stability Theorem

mphys(un) = max(0, n − 1)

For the silver-ratio sextic potential, the constrained Morse index of the n-th radial soliton mode is exactly max(0, n−1). This means: u0 is stable (electron), u1 is stable (muon), u2 is metastable with one decay direction (tau), and every higher mode has Morse index ≥ 2 (multiply unstable, no fourth lepton).

Fluctuation eigenvalue spectrum

Each row shows the lowest 8 eigenvalues of the fluctuation operator at that mode. Red bars are negative (unstable directions); green bars are positive (stable directions). The number of negative eigenvalues is the Morse index.

The Q-ball variational problem fixes the Noether charge Q (one constraint) and the soliton size (one dilation direction). After projecting these out, the physical Morse index is the count of remaining negative eigenvalues: this is what determines the actual stability of the soliton.

About this visualization

What's being shown

The dataset is a single run of family_of_three_theorem.py at the canonical published parameters: N = 3000 radial grid points, Rmax = 14, Coulomb coupling Z = 1.1, soft-core a = 1.145, quadratic confinement k2 = 0.039, ℓ = 0. The original run computes four radial modes u0, u1, u2, u3, and for each computes:

  • The profile un(r) — radial shape with n nodes.
  • The fluctuation operator's spectrum — the lowest several eigenvalues of the second variation of the energy at the mode. Negative eigenvalues mean unstable directions.
  • Two Morse indices: the raw count of negative eigenvalues, and the constrained count after projecting out the dilation direction (the direction of soliton-size rescaling) and the Noether-charge constraint.

Why constrained Morse index, not raw

The Q-ball is a constrained extremum of the energy: not a minimum in the full function space, but a minimum on the subspace of fixed Noether charge Q. The Hessian on the full space has spurious negative directions — including the dilation mode, which is always unstable for solitons in 3D (Derrick's theorem in disguise). What matters physically is the Morse index after these spurious directions are projected out.

After this projection, the constrained Morse indices are exactly (0, 0, 1, 2) for n = 0, 1, 2, 3. This matches max(0, n−1) — the F3 theorem prediction. u0 and u1 are stable; u2 has one decay direction (tau); u3 has two (excluded as a fourth lepton).

Why this isn't a live solver

Computing the F3 dataset at full resolution (N = 3000, four modes, full Hessian eigendecomposition with constraint projection) takes minutes on a workstation and tens of minutes in a browser via Pyodide. The result is also a one-shot proof: at the canonical silver-ratio parameters, the Morse-index pattern is (0, 0, 1, 2) — and that's the theorem. There's nothing meaningful to vary that would change the qualitative answer.

So this page presents the published canonical result rather than recomputing it. If you want to verify it yourself, run family_of_three_theorem.py from the corpus locally — the JSON output you'd produce is exactly what's visualised here.
